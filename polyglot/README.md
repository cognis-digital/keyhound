# Polyglot implementations

Parallel implementations of **keyhunt** (Scan firmware blobs and filesystem dumps for hardcoded private keys, API tokens, default creds, and weak RSA/ECC material.) across 9 languages, for reference and cross-language benchmarking.

- `python/` — python implementation
- `go/` — go implementation
- `rust/` — rust implementation
- `typescript/` — typescript implementation
- `c/` — c implementation
- `cpp/` — cpp implementation
- `java/` — java implementation
- `ruby/` — ruby implementation
- `csharp/` — csharp implementation
