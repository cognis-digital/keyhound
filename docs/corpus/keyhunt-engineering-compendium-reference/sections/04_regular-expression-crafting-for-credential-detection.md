RSA key moduli and exponents are typically represented in PEM or DER format, with PEM being more human-readable due to its base64 encoding and ASCII headers. In PEM format, the modulus and exponent are embedded within the public key section of the RSA structure. The modulus is a large integer, usually denoted as `n`, while the exponent, often referred to as `e`, is typically a small value like 65537. These values are encoded in base64 and are part of the larger ASN.1 structure that defines the key format.

To extract these values using regex, one must first identify the PEM-formatted section containing the RSA public key. The headers for RSA keys typically start with `-----BEGIN PUBLIC KEY-----` and end with `-----END PUBLIC KEY-----`.
